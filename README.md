## How to Use

Most of these are **copy-paste ready** with `[BRACKETED PLACEHOLDERS]` where you sub in your own context. A few are full system prompts — paste them at the start of a new session or into the system prompt field of whatever tool you're using.

No install, no setup. Just grab the template, fill in the blanks, and go.
