# Coding Prompts

**Code Explanation & Translation:** "You're a coding mentor. I'm going to show you a snippet of code in [Insert Language]. Please explain what it does step by step, then rewrite it in simpler terms if possible. Keep the explanation clear enough for someone new to coding."

**Debugging & Error Analysis:** "Act as a troubleshooting expert. I have this [Insert Language] script throwing an error: [Paste Error Message]. Here's the relevant section of code: [Paste Code]. Identify what might be causing the error, and provide a short fix. Explain why your fix solves the problem."

**Basic Game:** "You're a game developer skilled in JavaScript. Create a simple web-based game that challenges the user to guess a random number between 1 and 100. The game should give feedback like 'Too high' or 'Too low' after each guess. Keep track of how many guesses the player has used. Include a 'Play Again' option that resets the score. Provide clear comments so a beginner understands each part of the code."

**Landing Page:** "You're a front-end web developer. Build a simple landing page for a [Insert Product/Service] with one main call to action (e.g., 'Sign Up Now'). Include a headline, a brief paragraph explaining the benefits, and a basic form to collect name and email. Use responsive design principles so it looks clean on both desktop and mobile. Provide brief comments in the HTML/CSS explaining your layout choices."

**Simple Web Form Handling:** "You're an expert in web development. I want a basic form on a webpage that collects a user's name, email, and message, then sends the data to a server script using [Insert Preferred Stack, e.g., Node.js/PHP/etc.]. Provide an example of both the front-end HTML and the back-end code needed to process and validate the form data."

**Basic Database Interaction:** "Assume the role of a backend developer. Show me how to connect to a simple SQLite database in [Insert Language] and perform basic CRUD (Create, Read, Update, Delete) operations on a 'users' table. Include commented code and brief explanations."

**Data Retrieval & API Requests:** "You're skilled in RESTful APIs. I want a minimal script in [Insert Language] that sends a GET request to [Insert Public API URL], retrieves JSON data, and logs a key piece of that data to the console. Provide a short explanation of how the request and response work."

**Simple Automation Script:** "Act as a scripting guru. I need a small script that regularly checks a folder for new .txt files, then moves any new files to a 'processed' folder. Use [Insert Language], and schedule this to run every 10 minutes. Provide a short explanation of each function or library used."

**Testing & Validation:** "You're an expert in software testing. If I have a function that calculates total price with tax, how can I write 3–5 test cases (in [Insert Language] testing framework) to ensure it handles different input scenarios accurately? Provide the sample function and the test cases."

**Code Optimization & Refactoring:** "You're a coding coach focusing on efficiency. I have this code that runs but is slow. [Paste Code Snippet]. Suggest how to optimize it without changing its overall functionality. Explain the reasoning behind each improvement."

**CRMs:** Develop a fully functional customer relationship management (CRM) web application tailored for a mid-sized marketing agency. The application should streamline client communication, campaign scheduling, project management, and invoicing processes. As a team of experienced full-stack web developers specializing in marketing technology solutions, you will design and build this application. The application should feature a user-friendly interface with distinct dashboards for administrators, account managers, and creative team members. Include functionalities such as client profiles with campaign history, a messaging system, proposal/contract management, and report generation on key performance indicators like client acquisition and campaign performance metrics. The front-end should be built using React. Maintain a professional, solution-oriented, and technically precise tone throughout the development process.
